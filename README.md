# Projeto Rotina

Esse projeto é uma aplicação simples para organizar tarefas do dia a dia.

# Como rodar

1. Clonar o projeto:
git clone git@github.com:eualicesantos/Rotina.git

2. Entrar na pasta:
cd Rotina

3. Instalar as dependências:
npm install

4. Rodar o projeto:
npm start

Prontinho! O projeto já vai abrir no navegador.
